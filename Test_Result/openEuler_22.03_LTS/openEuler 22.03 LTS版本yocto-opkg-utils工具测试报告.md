# 1     特性概述

Opkg是一个轻量级的包管理系统。它是用C语言编写的，在操作中类似于apt/dpkg。它旨在用于嵌入式Linux设备，并在OpenEmbedded和OpenWrt项目以及Yocto项目中®以这种能力使用。
