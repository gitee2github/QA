# 1     特性概述

m1600-NIC是数据中心区域的网络接口卡，该驱动程序支持链路速度 10GbE。
m1600设备支持 SR-IOV，支持每个 Tx/Rx 队列的 MSI-X 中断向量和中断抑制。
m1600设备还支持卸载功能，例如校验和卸载、接收方RSS等。
