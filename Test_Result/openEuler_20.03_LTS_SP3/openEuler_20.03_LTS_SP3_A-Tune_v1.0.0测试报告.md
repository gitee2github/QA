# <br>概述

本测试报告为A-Tune v1.0.0在openEuler20.03-LTS-SP3操作系统上的测试报告，目的在于总结测试阶段发现的问题以及按版本及硬件平台测试结果，测试的范围主要包括安装、使用、稳定性等。
