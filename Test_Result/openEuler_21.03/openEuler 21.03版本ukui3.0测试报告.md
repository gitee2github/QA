Keywords 关键词：UKUI 测试报告

Abstract 摘要：按照测试要求，针对openEuler 21.03测试镜像对UKUI的安装、主要功能进行测试。测试结果良好，基本支持UKUI主要功能的正常使用。

# <br>概述

本测试报告为UKUI3.0在openEuler 21.03操作系统上的测试报告，目的在于总结测试阶段发现的问题以及按版本及硬件平台测试结果，测试的范围主要包括安装、使用、稳定性等。
