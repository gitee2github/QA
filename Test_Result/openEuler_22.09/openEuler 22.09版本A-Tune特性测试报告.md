# 1     特性概述

A-Tune是一款基于AI的操作系统性能调优引擎。A-Tune利用AI技术，使操作系统“懂”业务，简化IT系统调优工作的同时，让应用程序发挥出色性能。A-Tune提供在线调优和离线调优两种性能调优方式，并实现单机部署、分布式部署、集群部署等不同的部署方式供用户选择。

# 3     测试结论概述

## 3.1   测试整体结论

针对A-Tune特性及功能进行测试，共计执行38个测试用例，主要包含功能测试和集成测试，无skip用例，无失败用例，整体质量良好。

| 测试活动 | 活动评价 |
| -------- | -------- |
| 功能测试 | 38个针对应用的特性测试用例，覆盖分布式部署、集群部署下的A-Tune调优功能，aarch64及x86机器用例均全部通过 |

## 3.2   约束说明

- 分布式部署：使用两台同架构的机器进行测试
- 集群部署：使用三台同架构的机器进行测试
